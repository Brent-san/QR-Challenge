* Clean and minimal layout
* Centered card design
* Responsive styling

## 🛠️ Technologies Used

* CSS3

## 📁 Project Structure

* `index.html` – main HTML file
* `images/` – assets used in the project

This project was created as a practice exercise to improve my HTML and CSS layout skills.
